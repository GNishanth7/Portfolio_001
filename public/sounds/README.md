# Sound Files

This folder should contain the following sound effect files:

- `coin.mp3` - Coin insertion sound
- `select.mp3` - Menu navigation sound
- `confirm.mp3` - Action confirmation sound - Back/cancel sound
- `hover.mp3` - Button hover sound
- `start.mp3` - Game start sound
- `achievement.mp3` - Achievement unlock sound

## Recommended Sources for Free Retro Sound Effects

1. **Freesound.org** - Search for "8-bit", "retro", "arcade"
2. **OpenGameArt.org** - Free game assets including sounds
3. **BFXR** (bfxr.net) - Generate your own retro sound effects
4. **Mixkit.co** - Free sound effects library

## File Format

- Use MP3 or OGG format for best browser compatibility
- Keep files small (under 50KB each for SFX)
- Consider using mono audio for smaller file sizes
